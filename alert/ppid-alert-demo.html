Library menggunakan ikon dari Bootstrap Icons. Pastikan sudah di-include.
1. Dialog — Konfirmasi
PA.dialog()
Modal konfirmasi berbasis Promise — menunggu keputusan user
Promise<boolean>
Klik tombol untuk mencoba...
// Contoh penggunaan
PA.dialog({
  type : 'danger',    // 'info' | 'success' | 'warning' | 'danger' | 'question'
  title : 'Hapus Berita?',
  message: 'Data akan dihapus <strong>permanen</strong>.',
  confirm: 'Ya, Hapus',  // opsional — default per tipe
  cancel : 'Batal',     // false = tanpa tombol batal
}).then(result => {
  if (result) { /* konfirm */ }
  else { /* batal / ESC / klik luar */ }
});
2. Toast — Notifikasi Sudut
PA.toast()
Notifikasi kecil dengan progress bar, 6 pilihan posisi, pause on hover
Tipe
Posisi
PA.toast({
  type : 'success',
  title : 'Berhasil Disimpan!',
  message : 'Perubahan tersimpan ke database.',
  duration: 4000,    // ms — default 4000
  position: 'top-right' // 6 posisi
});
3. Alert — Inline di Halaman
PA.alert()
Alert yang diinjeksikan langsung ke elemen target, dapat di-dismiss
PA.alert({
  type : 'warning',
  title : 'Sesi Hampir Berakhir',
  message : 'Sesi Anda berakhir dalam 5 menit.',
  closable: true  // default true
}, '#alertTarget'); // selector CSS atau DOM element
4. Notify — Panel Notifikasi Besar
PA.notify()
Panel geser dari kanan — cocok untuk notif sistem & permohonan masuk
PA.notify({
  type : 'info',
  title : 'Permohonan Baru Masuk',
  message : 'Ahmad Fauzi mengajukan permohonan #PKL-2025-042.',
  duration: 6000 // ms
});
5. Loading — Overlay Proses
PA.loading()
Overlay loading — tutup manual lewat loader.close() atau PA.close()
// Tampilkan
const loader = PA.loading({
  title : 'Menyimpan Data...',
  message: 'Mohon tunggu sebentar.',
  dots : false  // true = dot pulse
});
loader.update('Mengunggah file...'); // update pesan
loader.close();                    // tutup
